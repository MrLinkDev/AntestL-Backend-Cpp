◆ is_connected()

bool TesartRbd::is_connected ( )
overridevirtual

Метод, возвращающий значение флага connected.

Предупреждения
Если был передан список адресов осей и с одной из осей не удалось установить соединение, то будет возвращён false.
Возвращает
Если все оси подключены - true. Если хотя бы одна из них не подключена - false.

Пример

RbdDevice *rbd = new TesartRbd("TCPIP0::localhost::5025::SOCKET;TCPIP0::localhost::5026::SOCKET");
if (!rbd->is_connected()) {
exit(1);
}

◆ next_angle()

int TesartRbd::next_angle ( int  axis_num)
overridevirtual

Переход к следующей угловой точке

Аргументы
[in]axis_numНомер оси ОПУ
Возвращает
Если угловая точка находится в пределах диапазона изменения угла, то возвращается ANGLE_MOVE_OK. Если угловая точка находится на границе диапазона, то возвращает ANGLE_MOVE_BOUND.

Пример

RbdDevice *rbd = new TesartRbd("TCPIP0::localhost::5025::SOCKET;TCPIP0::localhost::5026::SOCKET");
rbd.set_angle_range(-30, 30, 11, 0);
while (rbd->next_angle(0) != ANGLE_MOVE_BOUND) {
std::cout << "ОПУ повёрнута на следующую угловую точку" << std::endl;
}
