◆ wait_client()

int TcpServer::wait_client ( )

Функция, которая запускает прослушивание порта и ожидает клиента.

Предупреждения
Данную функцию следует вызывать только после вызова метода create().
Возвращает
Возвращает один из следующих возможных статусов:
  • SOCKET_NOT_LISTENING - не может быть запущено прослушивание
  • CLIENT_CONNECTED - клиент был подключен
  • CLIENT_NONE - нет клиента

Пример

TcpServer server();
int status = server.create();
if (status != SOCKET_OK) {
printf("Error");
return
}
status = server.wait_client();
if (status == CLIENT_CONNECTED) {
printf("Client connected");
}

◆ read_data()

int TcpServer::read_data ( std::string &  data)

Функция, осуществляющая чтение данных от клиента.

Данные записываются в переданный буфер. Буфер заполняется до тех пор, пока не встретятся символы, означающие конец посылки.

Аргументы
[out]data- Буфер для записи принятых данных.
Возвращает
Если данные приняты без ошибок, то возвращает DATA_READ_OK. В противном случае - DATA_READ_ERROR.

Пример

TcpServer server();
int status = server.create();
if (status != SOCKET_OK) {
printf("Error");
return
}
status = server.wait_client();
if (status == CLIENT_CONNECTED) {
string data{};
server.read_data(data);
printf("%s", data);
}

◆ send_data()

int TcpServer::send_data ( std::string  data)

Функция, осуществляющая отправку данных клиенту.

К передаваемым данным добавляется конец посылки, после чего они отправляются.

Аргументы
[in]data- Передаваемые данные.
Возвращает
Если данные отправлены без ошибок, то возвращает DATA_SEND_OK. В противном случае - DATA_SEND_ERROR.

Пример

TcpServer server();
int status = server.create();
if (status != SOCKET_OK) {
printf("Error");
return
}
status = server.wait_client();
if (status == CLIENT_CONNECTED) {
string data = "Test command";
server.send_data(data);
}
