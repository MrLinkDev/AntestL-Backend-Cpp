Метод, осуществляющий логирование

Данный метод сравнивает текущий уровень логирования с уровнем входящего сообщения. Если уровень входящего сообщения больше, чем текущий уровень, то сообщение игнорируется. Позволяет передать строку формата и аргументы для этой строки.

Аргументы
[in]levelУровень сообщения
[in]fmtФормат сообщения
[in]argsАргументы для сообщения

Пример

logger::log(LEVEL_INFO, "{} ({}) created", socket_tag, socket_port); // Сообщение будет проигнорировано
logger::log(LEVEL_ERROR, "{} ({}) no clients", socket_tag, socket_port); // Сообщение будет выведено
